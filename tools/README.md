# Tools

Internal utilities and secondary entrypoints live here so the repository root stays reserved for primary monorepo surfaces.

- `nexa.py`: Python implementation behind the root `nexa` launchers.
- `nexa_runtime.py`: shared path-resolution helpers for Nexa runtime utilities.
- `export_docs.py`: legacy full-doc export helper.
- `split_exports.py`: legacy split export helper.
- `unified_scraper.py`: ad hoc landing-page scraping and scoring utility.
- `sovereign_calendar.py`: local calendar MCP server prototype.
