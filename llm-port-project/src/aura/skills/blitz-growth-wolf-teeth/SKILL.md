## Objective

Create “wolf teeth” growth: sharp, repeatable distribution primitives that compound.

## Primitives (reusable)

- **Lead magnet artifact**: calculator, checklist, template, benchmark, teardown.
- **Social proof engine**: before/after, public changelog, testimonials pipeline.
- **Outbound batch**: scrape → enrich → personalize → send → follow-up.
- **Inbound hooks**: SEO “long-tail pain pages”, micro tools, embed widgets.
- **Referral loop**: invite-only beta + waitlist unlocks + rewards.

## Guardrails

- **Don’t spam**: opt-out, rate limits, relevance.
- **No credential scraping**: never harvest secrets/tokens.
- **Store consent**: keep minimal contact data, document sources.

## KPI tree

- **Top**: sessions, impressions
- **Mid**: CTR, opt-in rate
- **Bottom**: activation, conversion, CAC, payback, churn

## Automation targets

- Daily: scrape + enrich + queue 50–200 leads
- Daily: 1 post + 10–30 DMs + 1 artifact share
- Weekly: ship 1 feature + 1 case study + 1 partner outreach wave
