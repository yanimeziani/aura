# Transfer Documentation

This folder contains the formal transfer package for ULaval Corpus UL.

## Documents

- `ULAVAL_TRANSFER_PROTOCOL.md` - end-to-end transfer phases, controls, and completion criteria
- `TRANSFER_CHECKLIST.md` - operational readiness and acceptance checklist
- `OSS_SOURCE_REGISTER.md` - mandatory citation and licensing registry for OSS sources
- `HANDOVER_PACKAGE_INDEX.md` - canonical inventory of transfer artifacts
- `ACCEPTANCE_MEMO_TEMPLATE.md` - acceptance sign-off template for receiving authority

## Completion Rule

No transfer completion claim is valid until:
- all checklist items are complete
- OSS register has no `TODO` values
- acceptance memo is signed by both authorities
