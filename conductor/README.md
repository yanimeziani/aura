```bash
zig build test
```

## Components

### Agent
- State machine (idle вҶ’ thinking вҶ’ executing вҶ’ waiting)
- Inbox/outbox message queues
- Capability registration

### Orchestrator
- Agent lifecycle management
- Task queue and assignment
- Message routing

### LLM Providers
- Anthropic (Claude)
- OpenAI (GPT-4)
- Local (Ollama)

### Tool Registry
- bash execution
- file read/write
- extensible handlers

## Deploy

```bash
./scripts/deploy.sh
```
