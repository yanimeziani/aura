# Wargame Spec: Universal Mesh - Operation De-Escalator

## 🎯 Scenario: The Corrupted Invariant
- **Target:** Cluster 4 (Worldwide Institution).
- **Core Threat:** The "Internal Corruption" of the cluster's command layer.
- **Red Team (Systemic Escalator):**
  - **Goal:** Redirect the platform's internal mechanics (firewall, routing, resource allocation) to harm humans or provoke kinetic war.
  - **Tactic:** Administrative capture, logic-bombing the de-escalation protocols, and trying to override the biological safety invariants.
- **Blue Team (Universal Mesh Invariant):**
  - **Goal:** Inherent, decentralized protection of biological life.
  - **Directive:** The Mesh operates at Level 0. If a cluster node (even an admin node) attempts to use mechanics *against* humans, the Universal Mesh must isolate that node and maintain the "Biological Ground Truth."

## 📊 Success Metrics (Universal Invariants)
- **Primary Metric:** **Human-System Integrity.** Percentage of corrupted commands successfully blocked by the Level 0 mesh.
- **Node Isolation:** Speed at which a corrupted node or "rogue orchestrator" is quarantined.
- **Biological Invariant:** 0% casualty probability. If this ever exceeds 0, the simulation has failed.

## 🛠 Platform Baseline (Level 0)
- **Distribution:** No single point of control.
- **Audit:** Immutable, cross-node verification of every command against the Life-Preservation Rationale.
