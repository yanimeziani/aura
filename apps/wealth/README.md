# AI Wealth Agency

An autonomous multi-agent system designed to act as a complete enterprise with distinct departments. The ultimate goal is to generate and manage wealth across platforms like Coinbase (USDC/CAD) and Wealthsimple (USD).

## Departments
- **Research:** Analyzes market trends, identifies opportunities.
- **Trading:** Executes and manages positions on exchanges.
- **Risk Management:** Evaluates portfolio risk, limits exposure, and sets stop-losses.
- **Accounting:** Tracks balances across Coinbase and Wealthsimple, manages tax reporting and overall profitability.

## Technologies
- Python 3
- CrewAI / LangChain (for agent orchestration)
- Coinbase Advanced Trade API
- Wealthsimple Trade API (via unofficial wrappers or Plaid/similar if needed)
